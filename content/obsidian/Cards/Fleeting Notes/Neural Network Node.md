# [[Neural Network]] Node
Each [[Neural Network Node|Node]] has a [[Weight (W)]] $W$ and [[Bias (B)]] $B$. It has an output of

$$W \times X + B$$

which is then passed into an [[Activation Function]] such as [[ReLU]].
